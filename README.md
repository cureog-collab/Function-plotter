# C-based Function Plotter
A function plotter, basically Desmos on a budget, built in C.

## Compile and run
```bash
make
./build/function-plotter [function]
```

## Controls
still working, the plotter can now only render grids, principle X and Y axises.
The program supports dynamic grid drawing.

## Dependencies
* **SDL2**
* **tinyexpr**: **tinyexpr.c** & **tinyexpr.h**
